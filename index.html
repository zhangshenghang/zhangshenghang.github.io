hexo+github搭建

环境：Centos6.5

1.git基础安装
yum -y install git gcc gcc-c++ curl wget
2.安装nodejs
sudo yum install epel-release现在可以使用yum命令安装Node.js了。
sudo yum install nodejs因为在开发过程中我需要管理节点包，我还要安装新公共管理的软件包管理器，使用以下命令。
sudo yum install npm
3.安装hexo
npm install -g hexo-cli

注：默认下载比较卡，设置npm源，默认的比较卡
执行 npm config set registry “https://registry.npm.taobao.org“ 将npm包源指向淘宝，就不需要翻墙安包了

4.建站

安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。

$ hexo init <folder>
$ cd <folder>
$ npm install
5.启动

在folder目录下运行hexo server

默认使用4000端口，可以使用hexo server -p 5000 更改端口

至此可以内网使用了，使用hexo n “MyBlog”创建新文章

6.搭建github博客

新建一个名为你的用户名.github.io的仓库，比如说，如果你的github用户名是test，那么你就新建test.github.io的仓库（必须是你的用户名，其它名称无效），将来你的网站访问地址就是 http://test.github.io 了

创建完成后选择右侧的Setting页面

选择Choose a theme 选择一个主题进入，此时应该可以进入刚刚设置的http://test.github.io，会显示你选择的主题，

7.hexo绑定github

  • 配置Git个人信息
    如果你之前已经配置好git个人信息，请跳过这一个 步骤，直接来到

    1、设置Git的user name和email：(如果是第一次的话) 

    git config --global user.name "gitname"
git config --global user.email "xxxx@qq.com"

    2、生成密钥 

    ssh-keygen -t rsa -C "xxxx@qq.com"

  • 配置hexo

    deploy:
type: git
repo: git@github.com:yourname/yourname.github.io.git
branch: master

  • 配置git SSH keys


    选择SSH and GPG keys => new SSH key
    其中title随便起，key填入刚刚生成的秘钥内容，文件为 ~/.ssh/id_rsa.pub

8.发表文章

  • hexo n “myBlog”
  • hexo d -g //在部署前先生成
    执行以上两步即可在自己的域名下看到生成的文章了

踩坑提醒 

注意需要提前安装一个扩展：  
npm install hexo-deployer-git --save  
如果没有执行者行命令，将会提醒  
deloyer not found:git

有时候，复制文本（尤其是代码）到 Vim，会出现格式错乱的问题。看样子，应该是自动缩进惹得祸。

paste 模式
运行如下命令，进入 paste 模式：

:set paste
进入 paste 模式后，按 i 键进入插入模式，然后再粘帖，文本格式不会错乱了。

但粘帖后还需要按
进入普通模式并执行如下命令结束 paste 模式：
:set nopaste
